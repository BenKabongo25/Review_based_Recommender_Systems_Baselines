# LETTER

This repository provides the official PyTorch implementation of LETTER, as introduced in the following paper:

> **Rating-Aware Homogeneous Review Graphs and User Likes/Dislikes Differentiation for Effective Recommendations**

## Requirements

- Python 3.7
- Windows 10
- Torch 1.13.1
- CUDA 12.2
- numpy 1.21.6

## Usage
You can run the model with:

```bash
python model.py --dataset=<dataset>
```
